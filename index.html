Aggro-base

Laravel-AngularJS-D3.js visualization project

Project Details

  • Category : Projects
  • Authors : G. Salteris

Project description

The project build for the course Advanced Topics of Database Technology and Applications. The application uses agricultural data available from the world data bank regarding the agricultural trend in the European Union. For the project, we use laravel as the backend framework angular for the frontend and d3.js library for visualization.
